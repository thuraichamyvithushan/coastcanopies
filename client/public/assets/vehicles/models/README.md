Place real vehicle 3D model files here and reference them from the admin UI.

Example:

- `client/public/assets/vehicles/models/toyota-hilux.glb`
- `modelUrl` in admin: `/assets/vehicles/models/toyota-hilux.glb`

Use `modelScale`, `modelPosition`, and `modelRotation` in admin to align the mesh with the canopy scene.
`modelRotation` values are stored in degrees.
